Welcome to BROCR, the browser based OCR app!

BROCR uses Tesseract, Google's open source OCR tool. Apart from that, you also need pytesseract, a Python client to integrate Tesseract with Python. Other prerequisites include Pillow for image processing. Select the file you want to perform OCR in, select the language and hit upload. You will be redirected to another window containing the OCR text. Do keep in mind it might take some time to perform the OCR depending on the amount of text in the image.
